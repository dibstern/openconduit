Register a project with the daemon using the CLI: 
conduit project add \
  --name my-app \
  --path /home/user/projects/my-app \
  --port 4097
The daemon will mount a relay stack at /p/my-app and start watching for an OpenCode instance on the configured port. You can verify with conduit project list.
