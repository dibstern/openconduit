Martian Mono's wider character width creates a distinctive, unhurried rhythm. Each glyph occupies generous horizontal space — the typographic equivalent of a precision-milled aluminum case. Technical without being clinical. Playful without being casual. The wider set width means every word breathes.

Character Width

Martian Mono runs ~15% wider than JetBrains Mono and ~20% wider than Fira Code. This extra width gives conduit's interface a spacious, considered feel — each character sits in its own milled pocket.

Weight Range

Variable font from Thin (100) to ExtraBold (800). The wordmark uses Bold (700) for maximum presence. UI text uses Regular (400) and Medium (500) for comfortable reading at small sizes.
