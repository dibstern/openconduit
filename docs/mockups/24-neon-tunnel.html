06 — Chat
How does the relay broadcast SSE events to connected browser clients?

Each project’s RelayStack owns an SSE consumer that connects to the upstream OpenCode instance. Events are cached relay-side, then fanned out to all connected WebSocket clients:

async function broadcast(event: SSEEvent): Promise<void> { const payload = serialize(event); for (const client of this.clients) { client.send(payload); // fan-out to all } this.cache.append(event); }
