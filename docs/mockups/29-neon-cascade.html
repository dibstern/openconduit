Can you add fan-out logic so the SSE consumer broadcasts to all connected browser tabs simultaneously?

I'll add a fan-out layer to the SSE consumer. The key idea is a single upstream subscription that multiplexes events to all downstream WritableStream sinks:

class CascadeRelay { private sinks: Map<string, WritableStreamWriter> = new Map(); broadcast(event: SSEEvent) { for (const [id, sink] of this.sinks) { sink.write(event).catch(() => { this.sinks.delete(id); // prune dead channels }); } } }
