05 — Typography
Chakra Petch · Display / Wordmark
conduit
Ripples from the source
Chakra Petch brings Thai-influenced angles and a sci-fi edge to the wordmark. Its geometric construction mirrors machined metal — precise cuts with just enough character to feel alive. The echo mark radiates beside it, each ring carrying the signal further from the origin.
JetBrains Mono · Body / Code
Signal transmitted
The monospace face handles all operational text — status messages, timestamps, code, and interface labels. Clean and neutral against the angular display type.
relay.broadcast({ session: "abc-123", event: "echo", rings: 4 }) // outbound
Angular meets monospace
Chakra Petch speaks at the brand level — wordmark, headings, navigation. JetBrains Mono handles the operational layer — session status, timestamps, code fragments, and data. The contrast is sharp but controlled. Two typefaces, two neon colors, concentric rings expanding from a single point of origin.
