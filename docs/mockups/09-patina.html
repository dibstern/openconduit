The Unexpected Pairing — Serif Display + Monospace Body
A serif font for a developer tool is deliberately unexpected
Most dev tools reach for the same geometric sans-serifs. Conduit pairs an organic, variable serif with precise monospace — because the best interfaces, like the best materials, develop richer character through contrast and time. Fraunces' "wonky" axis gives letterforms an almost hand-drawn quality that feels alive against DM Mono's steady rhythm.
