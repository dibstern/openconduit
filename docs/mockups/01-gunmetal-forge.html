Body & Interface
Conduit is a web UI relay for OpenCode. It lets one long-lived relay daemon expose OpenCode sessions to browser clients across multiple projects. Each project gets its own relay stack, with session management, event streaming, and terminal access.
Code
// Start a relay daemon const daemon = await Daemon.create({ port: 2633, projects: [ { slug: "conduit", path: "/src/conduit" }, { slug: "myapp", path: "/src/myapp" }, ], }); await daemon.listen(); // Relay is live at http://localhost:2633/p/conduit
